OBS Setup Instructions

Browser Source Setup

  1. Add a Browser Source in OBS
  2. Set URL to: http://localhost:3000/display.html
  3. Set Width: 1080 and Height: 1920
  4. Check "Shutdown source when not visible" for better performance

Custom Dock Setup

For easy access within OBS, add this as a Custom Dock:

  1. In OBS, go to Docks → Custom Browser Docks...
  2. Dock Name: Caller Control
  3. URL: http://localhost:3000/dock.html
  4. Click Apply
  5. The dock will appear in your OBS interface
