Bible Verse Display

OBS Setup Instructions

Browser Source Setup

  1. In OBS, click the + button under Sources
  2. Select Browser
  3. Name it "Bible Verse Display" and click OK
  4. Set URL to: http://localhost:3000/bible-display.html
  5. Set Width: 1920 and Height: 1080
  6. Check "Shutdown source when not visible" for better performance
  7. Click OK

Custom Dock Setup (Optional)

For easy access within OBS, add this control panel as a Custom Dock:

  1. In OBS, go to Docks → Custom Browser Docks...
  2. Dock Name: Bible Verses
  3. URL: http://localhost:3000/bible-control.html
  4. Click Apply
  5. The dock will appear in your OBS interface

Tips

  • Use keyboard shortcuts for quick access: Enter to fetch & display, Escape to clear
  • Fetched verses are automatically cached for quick re-use
  • Multi-verse passages are split across pages (3 verses per page)
