# Application Icons

Place your application icons in this folder:

- `icon.png` - Main application icon (256x256 or larger)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `tray-icon.png` - System tray icon (16x16 or 32x32)

## Creating Icons

You can use tools like:
- [Electron Icon Builder](https://github.com/nicholasareed/electron-icon-builder)
- [IconConverter](https://iconverticons.com/online/)
- GIMP, Photoshop, or Figma

## Fallback

If no icons are provided, the application will use a simple colored fallback icon.
