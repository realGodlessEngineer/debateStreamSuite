Caller Display Setup

OBS Browser Source

  1. In OBS, click + under Sources
  2. Select Browser
  3. Name it "Caller Display" and click OK
  4. Enter the URL below
  5. Set Width: 1080 and Height: 1920
  6. Click OK

  1. In OBS, go to Docks → Custom Browser Docks...
